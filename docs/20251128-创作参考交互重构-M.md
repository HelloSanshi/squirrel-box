# 创作参考交互重构

## 📋 任务信息
- 类型：🟡 中等任务 (M)
- 预计时间：30 分钟
- 问题：原有交互"先在收藏库选卡片 → 再切到创作"很反人类

## 🎯 优化方案

### 原流程（反人类）
```
收藏列表选卡片 → 切到创作Tab → 开始写
```

### 新流程（符合直觉）
```
创作页 → 点击"添加参考" → 切到收藏列表（选择模式）→ 选完确认 → 回到创作页
```

## 交付

### 📝 代码变更
- `src/sidepanel/SidePanel.tsx` - 重构创作参考的交互流程

### 核心改动
1. **创作页面**：
   - 新增"创作参考"卡片区域，始终显示
   - 空状态：虚线框 + "从收藏中选择" 按钮
   - 已选状态：显示已选参考列表，可单独删除

2. **选择模式**：
   - `selectMode` 状态控制是否处于选择模式
   - 收藏列表顶部显示蓝色提示栏
   - 卡片右上角显示勾选框
   - 底部固定"确认选择"按钮

3. **交互优化**：
   - 非选择模式下，收藏卡片不可点击选中
   - 选择模式下点击卡片切换选中状态
   - 确认后自动返回创作页面
